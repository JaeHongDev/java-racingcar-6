# 자동차 경주 게임 요구사항 문서

## View

### MainView

- `ViewContainer`를 가지고 있으며 다른 `View`를 호출할 수 있습니다.

---

### 입출력 요구사항 정리

#### InputView

- [ ] 이름을 입력받습니다.
- [ ] 시도 횟수를 입력받습니다.

#### OutputView

- [ ] 실행 결과 메시지를 출력합니다.
- [ ] 최종 우승자를 출력합니다.

--- 

### 용어 정리

- player: 참가자
- car : 자동차
- winner: 우승자
- track: 자동차 트랙
